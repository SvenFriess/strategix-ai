Signal‑CLI: Nummer verknüpfen & Gruppe anlegen

So bindest du eine dedizierte Signal‑Nummer an den Bot und startest eine Gruppe.

Vorbereitung

  • macOS oder Linux
  • Java installiert (java -version)
  • Signal‑CLI installiert

1) Nummer registrieren

# Registrieren
signal-cli -u +49XXXXXXXXXX register

# Verifizieren (per SMS/TAN)
signal-cli -u +49XXXXXXXXXX verify 123456

# Geräte‑Name setzen (optional)
signal-cli -u +49XXXXXXXXXX updateProfile --name "Borgobatonebot"

2) Gruppe erstellen

# Neue Gruppe
signal-cli -u +49XXXXXXXXXX --verbose updateGroup --create --name "Borgo‑Bot" +49AAAAAA +49BBBBBB

# Gruppen‑ID anzeigen
signal-cli -u +49XXXXXXXXXX listGroups

3) Nachrichten empfangen/senden (Test)

# Empfangen (Timeout 5s)
signal-cli -u +49XXXXXXXXXX receive -t 5

# Senden
signal-cli -u +49XXXXXXXXXX send -m "Hallo aus dem Bot" +49AAAAAA

Tipp: Für den Produktivbetrieb nutze signal-cli daemon und ein System‑Service‑Script.
